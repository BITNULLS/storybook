# storybook
A motivational storybook that helps students learn.

## Directory Structure

 - `backend/`: Python Flask app to connect the database with the front end.
 - `docs/`: Documentation.
 - `edu-storybook/`: Front-end website (use `npm install` and `npm start` to run it)

## Requirements

 - Node >= 10.x
 - Python >= 3.x
